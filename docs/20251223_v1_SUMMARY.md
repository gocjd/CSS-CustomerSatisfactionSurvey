## 📦 납품 파일 구성 (3개 세트)

### 1️⃣ **20251223_v1_css.json** - 설문 데이터 정의
**용도**: 설문조사 시스템 구현 기반  
**크기**: ~8KB  
**내용**:
- 설문조사 기본 정보 (ID, 제목, 설명)
- 작성자 정보 (담당자, 부서, 연락처)
- 일정 정보 (날짜, 시간, 제외시간대, 시간대)
- 운영 설정 (익명성, 수정 가능 여부, 예상 소요시간 등)
- **3개 섹션** (기본정보, 서비스이용, 개선의견)
- **7개 질문** (Q1~Q7)
  - Q1: 연령대 (객관식/단일선택)
  - Q2: 이용빈도 (객관식/단일선택/음성프롬프트/조건분기)
  - Q3: 만족도 (객관식/Likert Scale)
  - Q4: 저이용 이유 (자유텍스트)
  - Q5: 미사용 이유 (음성의견)
  - Q6: 개선항목 (객관식/복수선택)
  - Q7: 최종의견 (음성의견)
- 각 질문별 검증 규칙, 점수, 분기 로직

**특징**:
✅ 확장된 스키마 (기본 설문 + 향상된 메타데이터)  
✅ 다국어 지원 준비 (ko, en, ja)  
✅ 오디오 메타데이터 포함  
✅ 분기 로직 구현 (Q2 → Q3/Q4/Q5)  
✅ 복수 선택 지원 (Q6)  
✅ 검증 규칙 상세 정의

### 2️⃣ **20251223_v1_desc.md** - UI/UX 화면 설계
**용도**: 프론트엔드 개발자용 UI/UX 명세서  
**크기**: ~15KB  
**내용**:
- 📋 목차 및 시스템 개요
- 🎯 사용자 여정 (User Flow)
- 📱 9가지 화면 설계
  1. 시작/로딩 화면
  2. 설문 소개 화면
  3. 텍스트 프롬프트 질문 (Q1, Q3, Q6)
  4. Likert Scale 질문 (Q3)
  5. 음성 프롬프트 질문 (Q2, Q5, Q7)
  6. 텍스트 의견 수집 (Q4)
  7. 음성 녹음 화면 (Q5, Q7)
  8. 복수 선택 질문 (Q6)
  9. 완료 화면
- 🎨 디자인 시스템 (색상, 타이포그래피, 여백)
- 📱 반응형 디자인 (모바일/태블릿/데스크톱)
- ♿ 접근성 지원 (WCAG AA)
- 🔒 보안 및 규정 준수
- ✅ 구현 체크리스트

**특징**:
✅ 각 화면마다 상세한 ASCII 다이어그램  
✅ 진행도 표시, 분기 로직 시각화  
✅ 음성 재생/녹음 상호작용 설명  
✅ 자동 저장 기능 명시  
✅ 다크모드 지원 방안
